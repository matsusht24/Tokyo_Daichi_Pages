オンライン処方に関する
よくあるご質問

Q

オンライン診療とはなんですか？

オンライン診療とは、インターネットと電話を用いて、診察を行い、お薬を郵送にて処方する診療方法です。 都度ご来院頂く必要がなく、空いた時間に診察を受けられるため、遠方の患者様やご来院のお時間を取れない方には大きなメリットとなります。

オンライン診療の際にも、電話問診を医師が直接行いますので、ご不明点やご不安なことがございましたら、医師に電話を通して直接ご質問頂けます。

Q

オンライン診療（郵送処方）の流れを教えて下さい。

アカウント作成 ➡ 電話問診のご予約 ➡ 電話問診 ➡ 決済・配送の選択 ➡ お薬が郵送で到着という流れです。

基本的には、一般的な通販サイトと同様の流れですが、医師との電話問診があることが唯一異なる点です。オンライン処方初回は、新規登録のお手間が少しございますが、2回目からは非常にスムーズにお薬の処方が受けられます。

Q

オンライン診療を利用するのに必要なものはなんですか

インターネットに接続されているパソコン・スマートフォン・タブレット端末が必要となります。また送受信可能なEメールアドレスが必要となります。

Q

パソコンを持っていないのですが、スマートフォンやタブレット端末からでも利用可能ですか？

インターネットに接続されているスマートフォンやタブレット端末からでも、ご利用頂けます。

Q

オンライン処方を利用できないbrowser(ブラウザ)はありますか？

iPhoneやAndroidの標準ブラウザでもある「Safari」や「Google Chrome」であれば問題ありませんが、スマホ広告をブロックするブラウザアプリ「Adblock Browser」をご使用の場合はシステムが正常に機能しなくなるため、ご利用いただけませんのでご注意下さい。
※「Mozilla Firefox」「Microsoft Edge」も利用可能です。

Q

直接来院した場合とは薬の値段は変わりますか？

お薬代は、ご来院された場合とまったく同じ価格なので、ご安心ください。

Q

送料はいくらかかりますか？

1回で１万円以上処方の場合には、送料は無料です。 処方合計額が1万円以下の場合の送料は、「送料とお支払い方法」をご確認ください。
※代金引換をご希望の方は、処方合計額に関わらず代引き手数料396円(税込)がかかります。
※銀行振込の場合、振込手数料は患者様のご負担となりますのでご了承下さい。

Q

薬代・送料以外でかかる費用はありますか？

電話でのオンライン診察の場合でも、診察料は無料です。かかる費用は、お薬代と送料のみです。
※1万円以上処方の場合は、送料無料です。
※代金引換をご希望の方は、代引き手数料396円(税込)がかかります。
※銀行振込の場合、振込手数料は患者様のご負担となりますのでご了承下さい。

Q

オンライン処方の場合は、薬はどうやって受け取るのですか？

手渡しの宅急便又は、ポスト投函にてお薬を郵送致します。郵便局のゆうパック又はゆうパケットを利用しております。 【ゆうパケット】
ポストに投函されます。手渡しの配達は行われません。「日時指定がない、かつ、荷物の厚みが1cm以下」の場合はゆうパケットにて配送いたします。送料は全国一律250円（税込）です。 【ゆうパック】
通常の宅急便です。手渡しでの配達が行われ受け取りのサインが必要で、日時指定が可能です。「日時指定がされた」または「荷物の厚みが1cm以上」の場合はゆうパックにて配送いたします。送料は地域によって異なります。「送料とお支払い方法について」をご確認ください。 ※どちらも発送後、ご登録されたメールアドレスに問い合わせ番号を添付して発送のご連絡を送信させていただきますので荷物の追跡は可能です。 ※お薬代10,000円以上（税込）は送料無料です。

Q

薬は何日くらいで届きますか？

配送先のご住所によりますが目安は決済後1～4日後です。※離島などはより日数がかかる可能性があります。

Q

土日や祝日をはさんだ場合は薬が届くまで日数がかかりますか？

電話でのオンライン診察の場合でも、診察料は無料です。かかる費用は、お薬代と送料のみです。
※1万円以上処方の場合は、送料無料です。
※代金引換をご希望の方は、代引き手数料396円(税込)がかかります。
※銀行振込の場合、振込手数料は患者様のご負担となりますのでご了承下さい。

Q

GW・お盆・年末年始といった大型連休中でも薬を送ってもらえますか？

【GWとお盆】
GWとお盆の時期は休みなく毎日診療しておりますのでオンライン処方可能です。
また、日本郵便のサービス「ゆうパック」「ゆうパケット」は365日毎日配達が行われておりますので、お薬のお届けも可能です。

【年末年始】
誠に申し訳ありませんが、年末年始の休診日は、オンライン診療もお休みさせて頂きます。余裕を持って診療予約をして頂けますと幸いです。

Q

診察券がなくなってカルテ番号も分からないのですが利用できますか？

利用可能です。
「アカウント新規登録」にてメールアドレス入力し送信後、ご入力いただいたアドレスにメールが届きます。 そのメールにある新規アカウント登録画面に移行するURLにアクセスしていただきカルテ番号は空白のまま、その他必要事項をご入力いただきますのでそれらの情報から該当するカルテを検索いたします。

Q

全く初めて（通院歴無し）なのですが利用できますか？

現在は、来院歴が無くても新規登録を受付けておりますので、ご利用いただけます。
オンラインでの問診票と電話問診にて現在の健康状態や使用中のお薬を確認させていただきます。
その際に医師が「来院していただき健康状態を確認しないと処方不可」と判断する場合もあることをご了承下さい。

Q

時間がないので今予約を入れてすぐに問診を受けたいのですが対応可能ですか？

ご対応可能です。
予約時間の選択画面で「今すぐ希望」という項目があるのでこちらをお選び下さい。 ※大変申し訳ございませんが、混み合っている時はお待ちいただく場合がございます。あらかじめご了承ください。
※診療時間の終了30分前からは予約は出来ません。

Q

全く初めて（通院歴無し）なのですが利用できますか？

「患者様のご本人確認」
　↓
「問診票に沿って現在の健康状態の確認」
　↓
「現在使用中のお薬の確認」
　↓
「特に問題がなければご希望のお薬と数量を聞いて終了」
となります。スムーズに進めば1～2分ほどで終わります。

Q

配送についてですが日時指定は可能ですか？

日時指定可能です。通常の宅急便（ゆうパック・手渡し）での発送となります。

Q

なかなか荷物の受け取りが出来ないのですが、いい方法はありますか？

発送荷物が1cm以下の場合は、ポスト投函（ゆうパケット）での発送が可能です。送料は日本全国一律250円（税込）です。 また外側からは薬と分からないように職場へお届けすることや、お近くの郵便局留をご選択頂くことも可能です。
※発送荷物が、1cm以上となる場合には、通常の宅急便での発送となります。

Q

職場に送ってもらうことは可能ですか？

送り先に該当患者様のお名前を明記する条件で可能です。

Q

マンションの宅配ボックスに届けてもらうことは可能ですか？

恐れ入りますが、委託の配送業者様（日本郵便）の範疇になりますので当院が対応可能とは申し上げられません。配送伝票の備考欄に「宅配ボックスへの配達希望」と書くことまでしかできないこと、ご理解ください。

Q

家族に知られないように受け取る方法はありますか？

(1) 配送元と配送品名を変更することができます。
※配送元「浜松町第一クリニック」⇒「HM商品センター」、配送品名「医薬品」⇒「筆記用具」に変更することが可能です。 (2) 配送先を「郵便局留め」にする方法があります。
ご希望の際は決済時の備考欄に「郵便局留め希望」の旨と、受け取りたい郵便局の「郵便番号」と「郵便局名」を必ずご入力ください。

Q

パートナー（女性）に送ってもらうことは可能ですか？

「〇〇様方」を了承してしまうと、問診を受けた患者様以外の方の誰にでもお送りすることが可能となってしまうため郵送できません。

Q

コンビニでの受け取りはできますか？

申し訳ございませんができかねます。
ご自宅への配送が難しい場合は「郵便局留め」をご利用ください。
ご希望の際は決済時の備考欄に「郵便局留め希望」の旨と、受け取りたい郵便局の「郵便番号」と「郵便局名」を必ずご入力ください。

Q

支払いは、どのようにするのでしょうか？

「クレジットカード」・「銀行振込」・「代金引換」の3つの方法があります。クレジットカードは［VISA、MASTER、JCB、DINERS、AMEX］がご利用いただけます。
※申し訳ございませんが、コンビニ決済や各種キャッシュレス決済は対応しておりません。

Q

クレジットカード決済の安全性は問題ないのでしょうか？

コンビニなどでも広く利用されている｢GMOイプシロン株式会社」 の決済代行サービスを利用しております。
クレジットカード決済を選択した場合、イプシロン決済画面へ移動いたします。
送信される際にはSSL（128bit）による暗号化通信で行い、クレジットカード情報は当院では保有せず、同社で厳重に管理されます。

Q

クレジットカードの明細にはどのように表記されますか？

「浜松町第一クリニック」と表記されます。

Q

電話問診後に薬の種類や数量を変更したいのですがどうすればよいですか？

お電話にて承ります。オンライン処方システム内（マイページ）で患者様側から変更することはできません。
